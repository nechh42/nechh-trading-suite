# Nechh Lab Bot Kurulumu

- Python 3.8+
- Pip paket yöneticisi

## 2. Kurulum
```bash
# Dosyaları indir
git clone https://github.com/nechh-lab/bot-repo.git

# Sanal ortam oluştur (Opsiyonel)
